Description:

Emac major mode for NAND hardward description language files (.hdl) as defined by the coursera class nand2tetris.

The mode provides

  • syntax / font-locking
  • customizable indentation: choose indentation for IN/OUT declarations, PARTS, and general offset.
  • Compilation support, running the chip in simulator, and jumping to mismatches in .out file.
  • Comparison buffer for expected vs. output. (C-c C-c)
  • imenu, other user functions, etc.

Tools in build directory: - Autogenerate snippets from the HDL survival website (for use with yasnippet) - Autogenerate docs.txt from builtin chips.
